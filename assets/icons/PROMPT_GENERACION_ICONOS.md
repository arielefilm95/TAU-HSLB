# PROMPT PARA GENERAR ICONOS E IMÁGENES - APP TAU

## CONTEXTO
Necesito generar iconos e imágenes para una aplicación PWA llamada "TAU" (Terapia de Audición y Lenguaje). La aplicación es una herramienta profesional para fonoaudiólogos que maneja registros de pacientes, madres, y seguimientos de terapia.

## REQUISITOS GENERALES
- Estilo: Profesional, médico/healthcare, limpio y moderno
- Paleta de colores: Azules profesionales (tonos médicos), blancos, grises suaves
- Temática: Audición, lenguaje, terapia, salud, tecnología médica

## ICONOS REQUERIDOS

### 1. ICONO PRINCIPAL (APP ICON)
- **Tamaños necesarios**: 72x72, 96x96, 128x128, 144x144, 152x152, 192x192, 384x384, 512x512 píxeles
- **Formato**: PNG con transparencia y SVG
- **Concepto**: 
  - Letras "TAU" estilizadas con elementos de audición (ondas sonoras, icono de oído)
  - Opción 2: Ícono que combine un oído estilizado con burbujas de diálogo
  - Opción 3: Diseño minimalista con forma de cabeza/perfil con ondas sonoras
- **Colores**: Azul profesional (#2C5282, #3182CE), blanco, gradientes sutiles

### 2. FAVICON
- **Tamaños**: 16x16, 32x32 píxeles
- **Formato**: PNG
- **Diseño**: Versión simplificada del icono principal

### 3. APPLE TOUCH ICON
- **Tamaño**: 180x180 píxeles (redondeado automáticamente por iOS)
- **Formato**: PNG
- **Estilo**: Diseño limpio que funcione bien con el redondeo de iOS

### 4. ICONOS ESPECÍFICOS DE LA APP (Opcional pero recomendado)
- **Registro de pacientes**: Ícono de perfil o carpeta médica
- **Dashboard**: Ícono de gráficos o panel
- **Reportes**: Ícono de documento o gráfico de barras
- **Configuración**: Ícono de engranaje
- **Tamaño**: 48x48, 96x96 píxeles
- **Formato**: PNG con transparencia

## IMÁGENES ADICIONALES REQUERIDAS

### 1. SPLASH SCREEN (PANTALLA DE CARGA)
- **Tamaños**: 
  - Móvil: 640x1136 (iPhone 5), 750x1334 (iPhone 6-8), 1242x2208 (iPhone 6+)
  - Tablet: 1536x2048 (iPad), 2048x2732 (iPad Pro)
- **Formato**: PNG
- **Contenido**: Logo TAU centrado, fondo degradado azul profesional
- **Texto**: "TAU - Terapia de Audición y Lenguaje" (opcional)

### 2. IMÁGENES DE CABECERA/HEADER
- **Tamaño**: 1200x300 píxeles
- **Formato**: PNG o JPG
- **Uso**: Para páginas principales del dashboard
- **Contenido**: Logo TAU con elementos gráficos sutiles de ondas sonoras

### 3. IMÁGENES PARA SECCIONES
- **Registro**: Imagen 400x300px relacionada con terapia o consulta médica
- **Reportes**: Imagen 400x300px con gráficos o datos médicos
- **Configuración**: Imagen 400x300px abstracta/profesional

## ESPECIFICACIONES TÉCNICAS

### FORMATOS
- **Iconos**: PNG (con transparencia) y SVG (para escalabilidad)
- **Imágenes grandes**: PNG (para calidad) o JPG (para optimización)
- **Resolución**: 72 DPI para web, 300 DPI para impresión (si aplica)

### OPTIMIZACIÓN
- **Tamaño de archivo**: Iconos < 50KB, Imágenes < 200KB
- **Compresión**: Sin pérdida de calidad visible
- **Metadatos**: Incluir información de copyright si es necesario

## GUÍA DE ESTILO VISUAL

### TIPOGRAFÍA
- **Principal**: Sans-serif moderna (Helvetica, Arial, similar)
- **Secundaria**: Tipografía limpia y legible

### COLORES RECOMENDADOS
- **Primario**: #2C5282 (Azul profesional)
- **Secundario**: #3182CE (Azul claro)
- **Acento**: #63B3ED (Azul muy claro)
- **Neutros**: #FFFFFF, #F7FAFC, #E2E8F0, #4A5568
- **Error/Alerta**: #E53E3E (Rojo suave)

### PRINCIPIOS DE DISEÑO
- **Minimalismo**: Limpio, sin elementos innecesarios
- **Profesionalismo**: Estilo médico/healthcare confiable
- **Accesibilidad**: Buen contraste, legibilidad
- **Consistencia**: Mismo estilo visual en todos los elementos

## REFERENCIAS VISUALES SUGERIDAS
- Aplicaciones médicas profesionales (Mayo Clinic, WebMD)
- Iconos de healthcare modernos y minimalistas
- Diseños de apps de terapia y salud mental
- Interfaz de sistemas médicos profesionales

## ENTREGABLES ESPERADOS
1. **Set completo de iconos** en todos los tamaños especificados
2. **Archivos SVG** originales para futuras modificaciones
3. **Imágenes de splash screen** para diferentes dispositivos
4. **Imágenes de cabecera** y secciones
5. **Guía de uso** con especificaciones técnicas

## NOTAS ADICIONALES
- Los iconos deben funcionar bien en fondos claros y oscuros
- Considerar versiones monocromáticas para ciertos usos
- Mantener coherencia con el branding existente de la aplicación
- Probar visibilidad en diferentes tamaños y dispositivos

## PROMPT PARA IA DE GENERACIÓN DE IMÁGENES

```
Crea un icono profesional para una aplicación médica llamada "TAU" (Terapia de Audición y Lenguaje). El diseño debe incluir:

1. Icono principal con las letras "TAU" estilizadas
2. Elementos visuales de audición (ondas sonoras, forma de oído estilizado)
3. Paleta de colores azules profesionales (#2C5282, #3182CE, #63B3ED)
4. Estilo minimalista y moderno, apropiado para healthcare
5. Fondo transparente
6. Alta calidad y escalabilidad

El icono debe transmitir profesionalismo, confianza y tecnología médica. Es para una aplicación PWA utilizada por fonoaudiólogos.

Genera también versiones en diferentes tamaños: 72x72, 96x96, 128x128, 144x144, 152x152, 192x192, 384x384, 512x512 píxeles en formato PNG con transparencia.
```

## PROMPT ALTERNATIVO (ENFOQUE DIFERENTE)

Diseña un logo iconográfico para "TAU - Terapia de Audición y Lenguaje" que combine:

- Un perfil de cabeza humana estilizado de lado

- Burbujas de diálogo representando el lenguaje
- Colores azules médicos con gradientes sutiles
- Diseño limpio y moderno
- Formato circular para mejor visualización en dispositivos

El estilo debe ser profesional pero moderno, similar a aplicaciones de salud como Headspace o Calm, pero con enfoque médico.
